== Description ==

Display a diary "Info Limousin" inserted into WordPress posts and pages.

To activate the diary, just write " [info-lim] "

----

Tapez le code pour afficher  :


- L'agenda du Limousin : [info-lim]

- L'agenda de la Corrèze [info-lim-19]

- L'agenda de la Creuse : [info-lim-23]

- L'agenda de la Haute-Vienne : [info-lim-87]

- L'agenda Expositions du Limousin : [info-lim-expo]

- L'agenda Fêtes du Limousin : [info-lim-fete]

- L'agenda Formation du Limousin : [info-lim-form]

- L'agenda Jeux du Limousin : [info-lim-jeu]

- L'agenda Marchés du Limousin : [info-lim-march]

- L'agenda Nature du Limousin : [info-lim-natur]

- L'agenda Spectacles du Limousin : [info-lim-spect]

- L'agenda Sport du Limousin : [info-lim-sport]



== Installation ==

1. Download info-limousin.zip from http://www.asso.info-limousin.com/

2. Decompress the file contents

3. Upload the info-limousin folder to your WordPress plugins directory (/wp-content/plugins/)

4. Activate the plugin from the WordPress Dashboard

5. See instructions for using the plugin
